# SelfWritingAI
 A system for writing AI code, or having an AI write AI code.
 
 ## How to use
 #### Commands
 runpong
 > Runs the game pong
 
 runmario
 > Runs the game mario
 
 runbattle
 > Runs a multiplayer battle simulation
 
 learnpong
 > Runs the program that learns how to build AIs to play pong.
